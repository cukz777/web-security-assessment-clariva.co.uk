# Web Security Assessment – clariva.co.uk

## Overview

Basic web security assessment of a live website.

---

## Scope

* Target: https://www.clariva.co.uk
* Type: Passive reconnaissance

---

## Key Findings

* WordPress (frontend exposure)
* No public login panel
* Limited attack surface
* robots.txt reveals /wp-admin

---

## Conclusion

Low-risk application with minimal exposure.

---

## Skills Demonstrated

* Reconnaissance
* Enumeration
* Security analysis

* ## Lessons Learned

* Not all applications expose exploitable vulnerabilities
* robots.txt is not a security mechanism
* Reducing attack surface significantly improves security posture
